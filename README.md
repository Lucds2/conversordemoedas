# 💸 Conversor de Moedas

Uma aplicação web interativa e responsiva desenvolvida para realizar a conversão de valores entre diferentes moedas globais em tempo real. O sistema conta com uma interface intuitiva, atualização dinâmica de bandeiras e suporte a moedas fiduciárias e criptoativos.

---

## 🚀 Funcionalidades

* **Conversão Multimoedas:** Suporte completo para conversões entre **Real (R$)**, **Dólar Americano (US$)**, **Euro (€)**, **Libra Estelina (£)** e **Bitcoin (₿)**.
* **Interface Visual Dinâmica:** Alteração automática das bandeiras e nomes dos países de origem e destino conforme a seleção do usuário.
* **Formatação em Tempo Real:** Exibição dos resultados formatados de acordo com a moeda selecionada, garantindo uma experiência de usuário limpa e profissional.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento Front-End puras (Vanilla Architecture):

* **HTML5:** Estruturação semântica da aplicação (`index.html`), utilizando seletores estruturados (`select`, `option`), campos de entrada otimizados (`input`) e carregamento de scripts no encerramento do escopo do documento.
* **CSS3:** Estilização moderna e layout responsivo (`style.css`), com suporte à importação de fontes externas (Roboto via Google Fonts).
* **JavaScript (ES6):** Manipulação avançada do DOM (`script.js`) para capturar eventos de clique, mudanças de seleção e processamento lógico dos cálculos de câmbio.

---

## 📂 Estrutura de Arquivos

* `/assets`: Diretório contendo os recursos visuais do projeto (Logotipo em GIF, vetores e as imagens de bandeiras dos países).
* `index.html`: Estrutura de marcação e elementos da interface do conversor.
* `style.css`: Folha de estilo responsável pelo design, posicionamento e identidade visual.
* `script.js`: Arquivo contendo a lógica de negócios, cálculo de conversão e atualização dos elementos visuais.
